🛒 Amazon Clone (E-Commerce Frontend Project)

📌 Project Overview

This project is a fully responsive **Amazon Clone** built using **HTML, CSS, and JavaScript**, designed to replicate the core UI and user experience of the Amazon website. It demonstrates strong frontend development skills, responsive design, and interactive UI components.

---

🚀 Features

* 🏠 **Homepage UI** similar to Amazon with navbar, banner, and product sections
* 🔍 **Search bar UI** with category selection
* 🛍️ **Product Listing & Product Detail Pages**
* 🛒 **Cart UI Functionality** (add/remove items interface)
* 🔐 **Login & Signup Pages UI**
* 🎞️ **Image Slider (Carousel)** for banners using JavaScript
* 📦 **Scrollable Product Sections** with smooth horizontal scrolling
* 🌐 **Responsive Design** (Mobile + Tablet + Desktop support)

---

🛠️ Tech Stack

* **HTML** – Structure of the website
* **CSS** – Styling and layout (Flexbox, Media Queries)
* **JavaScript** – Interactivity and dynamic behavior

---

📱 Responsiveness

The website is fully responsive and optimized for different screen sizes using **Flexbox** and **Media Queries**, ensuring a seamless user experience across devices.

---

🎯 Purpose of the Project

* Practice frontend development skills
* Understand real-world e-commerce UI design
* Improve JavaScript-based interactivity
* Build a strong portfolio project for placements
